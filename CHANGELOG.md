# 更新记录 / Changelog

## [2026-04-21] 功能增强版

### 新增功能

#### 1. 单通道/双通道模式切换
- **通道模式选择**：在「本地音频」配置卡片中新增模式选择
  - **单通道模式**（默认）：仅需麦克风，无需虚拟声卡
  - **双通道模式**：需要 VB-Cable 虚拟声卡，支持双向翻译
- **自动界面适配**：单通道模式下翻译面板占据全宽，双通道模式左右分栏

#### 2. TTS 语音输出开关
- 新增「语音输出」复选框控制是否播放翻译后的语音
- 关闭 TTS 后仅显示文字，适合仅需文字翻译的场景
- 禁用 TTS 时自动隐藏音色选择选项

#### 3. 独立显示窗口
- 新增「显示窗口」功能，可在独立窗口展示翻译结果
- 支持通过 URL 参数传递样式设置，避免闪烁问题
- 显示窗口特性：
  - 双栏显示：上栏译文，下栏原文
  - 可自定义：字体大小、行高、文字颜色、背景颜色
  - 设置自动保存到 localStorage
  - 支持实时同步主窗口的翻译内容

#### 4. 主界面文字样式自定义
- 新增「字体设置」面板（点击 🎨 按钮）
- 可调节：
  - 文字大小（10px - 32px）
  - 译文颜色
  - 原文颜色
  - 背景颜色

### 优化改进

#### 技术优化
- **URL 参数传递设置**：显示窗口通过 URL 参数 (`?fs=24&tc=ffffff&bc=000000`) 接收初始样式，避免 localStorage 读取延迟导致的闪烁
- **CSS 变量动态更新**：使用 CSS 变量实现样式实时切换，无需刷新页面
- **SessionStorage 配置持久化**：通道模式和 TTS 设置保存在 sessionStorage，页面刷新后保持

#### 界面优化
- 单通道模式下隐藏虚拟声卡状态检测区域
- 双通道模式保持完整功能不变

### 使用场景示例

#### 场景1：英文演讲实时翻译（英译中）
配置：
- 语言方向：我说的 → 英语，对方听到的 → 中文
- 通道模式：单通道
- TTS 输出：关闭
- 无需虚拟声卡

#### 场景2：双语会议（中英互译）
配置：
- 通道模式：双通道
- 安装 VB-Cable A+B
- TTS 输出：开启（可选）
